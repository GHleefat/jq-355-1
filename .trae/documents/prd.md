## 1. 产品概述
一款沉浸式 3D 滑翔机飞行体验网页应用。玩家驾驶滑翔机在云海与山谷间穿行，利用上升气流保持高度，享受纯粹放松的飞行乐趣，也可选择收集星星模式获得轻度游戏目标。

- 核心价值：提供无压力的沉浸式飞行放松体验，兼具轻度探索收集玩法
- 目标用户：追求放松、喜欢自然风景与飞行体验的所有用户

## 2. 核心功能

### 2.1 功能模块
1. **主菜单页**：模式选择、操作说明、开始入口
2. **飞行场景页**：3D 地形、体积云、滑翔机物理、上升气流系统、HUD 显示
3. **暂停/设置面板**：暂停游戏、切换模式、调整视角

| 页面名称 | 模块名称 | 功能描述 |

## 3. 核心流程

用户打开页面 → 进入主菜单，欣赏动态氛围背景 → 选择飞行模式（放松/收集星星） → 点击开始 → 进入 3D 飞行场景 → 鼠标控制滑翔机俯仰转向 → 寻找并进入上升气流维持高度 → 欣赏云海山谷风景（收集星星可选） → ESC 暂停或继续飞行 → 返回主菜单。

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：天空蓝渐变（#87CEEB → #1E3A5F），云白（#F0F8FF），山谷深绿（#2D4A3E）
- **强调色**：金色（#FFD700）用于星星与气流高亮
- **字体**：标题使用 Cormorant Garamond（优雅衬线），正文使用 Manrope（现代无衬线）
- **UI 风格**：半透明毛玻璃效果，极简 HUD，无多余装饰，突出沉浸感
- **按钮**：圆角胶囊按钮，半透明白底 + 白色描边，悬停时发光

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 主菜单页 | 标题区 | 大号衬线字体标题"云端滑翔"，副标题小字，居中布局，背景动态云层流动 |
| 主菜单页 | 模式选择 | 两张半透明卡片并列，左侧放松模式（云图标），右侧收集模式（星图标），选中时有金色描边发光 |
| 主菜单页 | 开始按钮 | 大号胶囊按钮，底部居中，悬停上浮并发光 |
| 主菜单页 | 操作说明 | 右下角小字区，鼠标图标+文字说明 |
| 飞行场景页 | HUD 左上角 | 高度计、速度表，白色半透明文字 |

### 4.3 响应式
桌面优先，鼠标控制为核心交互方式。移动端降级为触屏虚拟摇杆（简化版），保证基本可玩。

### 4.4 3D 场景指引
- **环境与氛围**：日落/黄昏时分的天空，橙红色地平线过渡到深蓝高空，体积云被阳光染色
- **光照设置**：单一方向主光源（太阳）+ 半球环境光，启用软阴影，云体积散射
- **摄像机**：第三人称尾随视角，跟随滑翔机后方略高处，平滑跟随，轻微晃动增强沉浸感
- **构图要素**：连绵起伏的程序化山脉占据画面下半部，中上部是流动的体积云层，滑翔机位于画面中央偏下的黄金分割点
- **交互动画**：滑翔机倾斜转向时机身自然侧倾，穿云时有雾化粒子效果，进入气流时光晕脉冲
- **后处理**：Bloom（发光）、轻微 Vignette（暗角）、ACES 色调映射、雾效深度
- **性能预算**：目标 60fps，地形使用低模 LOD，云层使用精灵片层模拟体积，draw call < 200
